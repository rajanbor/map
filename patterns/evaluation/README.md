# Evaluation Patterns

**Focus:** measuring quality, catching regressions, and detecting hallucination — how to know whether an AI system actually works.

Patterns in this category follow the [MAP pattern template](../_template/PATTERN_TEMPLATE.md).
See the [Roadmap](../../ROADMAP.md) for the full backlog and status.

> No patterns published yet. Want to write one? Open a **"New Pattern"** issue and see
> [CONTRIBUTING.md](../../CONTRIBUTING.md).

## Planned patterns

- Golden Dataset
- LLM-as-Judge
- Regression Testing
- Hallucination Detection
- Rubric Scoring
- Pairwise Comparison
- Faithfulness / Groundedness Evaluation
- Synthetic Eval-Data Generation
- Online A/B Testing

## Contributing to this category

1. Claim a pattern above (or propose a new one) via a **New Pattern** issue.
2. Copy the [template](../_template/PATTERN_TEMPLATE.md) to `evaluation/<pattern-slug>/README.md`.
3. Open a PR into `dev`.
