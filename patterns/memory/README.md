# Memory Patterns

**Focus:** persisting and recalling state across turns and sessions — from simple conversation buffers to long-term, episodic, and knowledge-graph memory.

Patterns in this category follow the [MAP pattern template](../_template/PATTERN_TEMPLATE.md).
See the [Roadmap](../../ROADMAP.md) for the full backlog and status.

> No patterns published yet. Want to write one? Open a **"New Pattern"** issue and see
> [CONTRIBUTING.md](../../CONTRIBUTING.md).

## Planned patterns

- Conversation (Buffer) Memory
- Sliding-Window Memory
- Summary Memory
- Long-Term Memory
- Episodic Memory
- Semantic Memory
- Knowledge-Graph Memory
- Entity Memory
- Vector-Backed Memory
- Memory Consolidation & Forgetting

## Contributing to this category

1. Claim a pattern above (or propose a new one) via a **New Pattern** issue.
2. Copy the [template](../_template/PATTERN_TEMPLATE.md) to `memory/<pattern-slug>/README.md`.
3. Open a PR into `dev`.
