# Website (Future)

This directory is a placeholder for the MAP documentation website.

The vision is an interactive, searchable guide that lets developers navigate the
catalog **by the question they're asking** — "RAG or fine-tuning?", "which retrieval
strategy?" — and jump straight to the relevant patterns and decision guides.

## Planned scope

- Render the Markdown catalog in [`../patterns/`](../patterns/) as a browsable site.
- Full-text and semantic **search** across patterns.
- **Decision trees** that link questions to patterns.
- Per-pattern rendering of Mermaid diagrams and code.
- Community translations.

## Status

Not started. The catalog is authored as plain Markdown so it remains the single source
of truth regardless of which site generator we adopt. Tooling choice will be decided in
a **Discussion** before implementation.

See the [Roadmap](../ROADMAP.md), Phase 3.
